# learn-agentic-patterns

Hands-on tutorials that teach **harness engineering** — shaping an AI coding assistant with CLAUDE.md, skills, conventions, and constraints so its output matches how your team actually works.

Every tutorial is a small runnable app. You run the same driving prompt (e.g. "build a todo app") against two versions of the scaffold: one bare, one with a harness. The difference in output is the lesson.

## Get started

Tutorials are grouped by skill domain. Pick whichever patterns you want to learn — anyone can follow any domain:

- [engineering/](engineering/) — harness engineering patterns for software development work.
