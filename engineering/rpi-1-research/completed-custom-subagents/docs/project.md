# Project

A lightweight personal todo app. Helps an individual capture, organise, and
complete their own tasks on a single device.

## Users

One person managing their own work. No multi-user, no sharing, no teams.

## Scope

- Create, edit, complete, delete tasks
- Order tasks (priority or manual)
- Persist across sessions

## Non-goals

- Collaboration, sharing, or accounts
- Native mobile apps (web only)
- Calendar, reminders, or notifications (may come later)

## Shape

Single-page web app (Vite + React) backed by a small HTTP API (FastAPI).
State lives server-side; the frontend is a thin client.
