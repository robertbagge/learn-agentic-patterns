# Iconography

## Icon Family

**Lucide** -- outlined stroke icons with consistent 1.5px weight. Rounded joins and caps.

## Icon Sizes

| Size | Name | Usage |
|------|------|-------|
| 12px | Micro | Inline indicators |
| 14px | Small | Button icons, inline actions |
| 16px | Default | Navigation, standard actions |
| 18px | Medium | Prominent buttons |
| 20px | Large | Toast icons, feature icons |
| 24px | XL | Empty states, hero elements |

## DOs and DON'Ts

**DO:**
- Always pair icons with text labels in navigation
- Use consistent sizing within a component
- Use semantic colours for status icons

**DON'T:**
- Never use icons as the sole way to communicate meaning
- Never mix icon families (stick to Lucide only)
- Never scale icons to non-standard sizes
