# building-blocks-1-skills

A layered **skills** stack on top of `rpi-1-full-cycle/completed-custom-subagents`.
Six skills spanning infra / capability / orchestration wrap a commit + PR
workflow — the intended walkthrough material for the skills architecture
session. Read the SKILL.md descriptions, classify each by layer, then go
build analogous skills for your own repo.

- [`starter/`](starter/) — the scaffold + all six skills, ready to walk through or run.
- [`completed/`](completed/) — identical copy of `starter/` for now; placeholder for a learner-task target.

Start in [`starter/`](starter/README.md).
