# Typography

## Font Families

| Font | Variable | Usage |
|------|----------|-------|
| Sora | `--font-display` | Headlines, metrics, section titles |
| Inter | `--font-body` | Body text, labels, buttons, navigation |
| Space Mono | `--font-mono` | Percentages, data indicators, code |

## Type Scale

| Role | Size/Line-height | Letter-spacing | Font |
|------|-----------------|----------------|------|
| Page Title | 38 / 1.0 | -1.5 | Sora Bold 700 |
| Metric Value | 36 / 1.0 | -2 | Sora Bold 700 |
| Section Title | 20 / 1.0 | 0 | Sora Semibold 600 |
| Card Title | 16 / auto | -0.5 | Sora Bold 700 |
| Body Strong | 14 / 1.5 | 0 | Inter Semibold 600 |
| Body Default | 14 / 1.5 | 0 | Inter Normal 400 |
| Label | 13 / auto | 0 | Inter Medium 500 |
| Caption | 12 / auto | 0 | Inter Medium 500 |
| Section Label | 11 / auto | +2 | Inter Bold 700 UC |
| Micro | 10 / auto | 0 | Inter Medium 500 |

## DOs and DON'Ts

**DO:**
- Use Sora for display and emphasis only
- Use Inter for all functional text
- Use uppercase + wide tracking for section labels only
- Maintain negative letter-spacing on large display text

**DON'T:**
- Never use Sora for body paragraphs
- Never use more than 2 font weights on a single surface
- Never go below 10px for any text
- Never use positive letter-spacing on display sizes
